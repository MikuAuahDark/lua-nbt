◆ inputReader

typedef string(* nbt::inputReader) (integer minread)

Input reader function signature.

When function is supplied to nbt::decode(inputReader, string) these things must be considered:

  • Returns uncompressed data.
  • Returns at least minread length of binary-string. If it returns less than that, the decoding function will fail. Function is allowed to return size bigger than minread.
  • If error occured, return nil AND the reason in string (2 values).
Parameters
minreadMinimum string that is needed by the decoder.
Returns
Uncompressed binary-string of the next NBT data needed.

◆ newCompound()

Tag nbt::newCompound ( table  value,
string  name = "" 
)

Create new TAG_COMPOUND NBT tag.

Warning
This function does not support automatic Tag type conversion, so the value must be a type of Tag.
Exceptions
errorthrown if it contains value other than Tag type.
-- Example of creating TAG_COMPOUND
local nbt = require("nbt")
tagCompound = nbt.newCompound({
hello = nbt.newString("World"),
integerValue = nbt.newInt(12345),
doublePrecision = nbt.newDouble(math.pi),
list = nbt.newList(nbt.TAG_STRING, {"bandori", "bang", "dream", "livesim2", "over the rainbow"})
})
