Encode the current NBT tag to its binary representation.

Parameters
noprefixInternal use. Always omit this when calling this function.
Warning
Care must be taken when calling this for type other than TAG_COMPOUND. Some NBT parsers only decodes NBT data if it's begin with TAG_COMPOUND. This library is not one of them.

◆ getValue()

any nbt::Tag::getValue ( )

Get the value of the NBT tag.

  • Number is returned for TAG_BYTE, TAG_SHORT, TAG_INT, TAG_LONG, TAG_FLOAT, and TAG_DOUBLE type.
  • String is returned for TAG_STRING type.
  • Table is returned for TAG_LIST, TAG_COMPOUND (key-value pairs), TAG_BYTE_ARRAY, TAG_INT_ARRAY, and TAG_LONG_ARRAY type.
